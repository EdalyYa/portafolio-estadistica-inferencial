Unidad I · Fundamentos de Inferencia

1) Muestreo y distribuciones

  • Población vs muestra. Parámetro vs estadístico.
  • Métodos de muestreo: aleatorio simple, sistemático, estratificado.
  • Distribuciones clave: Normal, t-Student, Chi-cuadrado, F.
  • Teorema del Límite Central y sus implicancias.

2) Estimación puntual

  • Media muestral, proporción muestral, varianza.
  • Sesgo y eficiencia; error estándar.
Diagrama conceptual: poblaciones, muestras y estadísticos.

3) Intervalos de confianza

  • Para media con σ conocida: \( IC = \bar{x} \pm z_{\alpha/2}\, \tfrac{\sigma}{\sqrt{n}} \)
  • Para media con σ desconocida: usa t-Student (\( n-1 \) gl).
  • Para proporción: \( IC = \hat{p} \pm z_{\alpha/2}\, \sqrt{\tfrac{\hat{p}(1-\hat{p})}{n}} \)
  • Tamaño de muestra para media y proporción.

4) Buenas prácticas

  • Verificar supuestos (normalidad, independencia).
  • Reportar nivel de confianza y método usado.

Unidad I · Ejercicios guiados

Ejemplo: Para una muestra de tamaño n=36 con \bar{x}=52 y \sigma=9, calcule el IC del 95% para la media.

  1. z_{0.025}=1.96
  2. Error estándar: 9/\sqrt{36}=1.5
  3. Margen: 1.96×1.5=2.94
  4. IC: 52±2.94 ⇒ [49.06, 54.94]

Interpretación: Con 95% de confianza, la media poblacional se encuentra en ese intervalo.
