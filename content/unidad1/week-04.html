Semana 4 · IC media (σ conocida) (I)

IC para la media con \(\sigma\) conocida
IC media (Z): \( \bar X \pm z_{\alpha/2}\,\tfrac{\sigma}{\sqrt{n}} \)

\[ IC_{1-\alpha}:\ \ \bar X \pm z_{\alpha/2}\,\frac{\sigma}{\sqrt{n}} \]

Interpretación: si repitiéramos el muestreo muchas veces, el \((1-\alpha)\) de los intervalos contendrían \(\mu\).

Gráfico del intervalo alrededor de la media.

Semana 4 · Tamaño de muestra (II)

Determinación de tamaño de muestra
Media: \( n = \left( \tfrac{z_{\alpha/2}\,\sigma}{E} \right)^{2} \)

Objetivo de margen de error \(E\): \[ n = \left( \frac{z_{\alpha/2}\,\sigma}{E} \right)^2 \]

Corrección por población finita: \( n_F = \tfrac{n}{1 + n/N} \).

Ejercicio guiado

Calcular \(n\) para \(E=2\), \(\sigma=9\), confianza 95%.

\(z_{0.025}=1.96\). Entonces \(n=(1.96\cdot 9/2)^2\approx 77.4\Rightarrow n=78\). Si \(N=2{,}000\), ajustar: \(n_F=\tfrac{78}{1+78/2000}\approx 75\).

Semana 4 · Tabla de n (III)

Tabla de tamaño de muestra (Media y Proporción)
Parámetro N conocida N desconocida
Media \[ n = \frac{Z^{2}\,S^{2}\,N}{N\,e^{2} + Z^{2}\,S^{2}} \] \[ n = \frac{Z^{2}\,S^{2}}{e^{2}} \]
Proporción \[ n = \frac{Z^{2}\,p\,q\,N}{N\,e^{2} + Z^{2}\,p\,q} \] \[ n = \frac{Z^{2}\,p\,q}{e^{2}} \]

Convenciones: \(Z\) cuantíl normal estándar; \(S\) desvío muestral (o \(\sigma\) estimada); \(e\) error máximo permitido; \(q=1-p\).

Guía rápida para decidir \(n\).
